# project-20
- is a web-based system for Kiliog Elementary School project-20 tracker.
- project 20 provides financial aid to a deceased teacher’s family through pooled contributions, fostering unity and compassion in DepEd Bukidnon.

## Features
- Project tracking
- Progress monitoring
- Reporting and analytics

## Tech Stack
- Frontend: React
- Backend: Node.js
- Database: MySQL


## Project Structure

```text
project-20/
├── client/                # React Frontend (Vite)
│   ├── public/            # Static assets
│   └── src/
│       ├── assets/        # Images, icons
│       ├── components/    # Reusable UI components
│       ├── context/       # State management
│       ├── hooks/         # Custom React hooks
│       ├── layouts/       # Page layouts
│       ├── pages/         # Page components
│       ├── services/      # API communication

│       └── utils/         # Helper functions
└── server/                # Node.js Backend
    ├── src/
    │   ├── config/        # DB & env configuration
    │   ├── controllers/   # Route handlers
    │   ├── middleware/    # Auth & error handling
    │   ├── models/        # DB interactions
    │   ├── routes/        # API endpoints
    │   └── utils/         # Helper functions
    └── .env               # Environment variables
```
